Perform a CO2 meassuring

In this use case, a CO2 meassuring will be performed.

Note
The driver must be initialised Initialise the driver before a measuring can be performed.
  1. Define a variable to get the CO2 ppm in and a return code variable.
    uint16_t ppm;
    mh_z19_return_code_t rc;
  2. Ask the driver to perform measuring.
    rc = mh_z19_take_meassuring();
    if (rc != OK)
    {
    // Something went wrong
    }
  3. When the driver has received the new ppm value from the sensor the speciefied call back function (see Initialise the driver) will be called.
