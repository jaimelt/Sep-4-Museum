Detailed Description

Honeywell HumidIcon Digital Humidity/Temperature Sensor HIH8220 driver.

The user manual for HIH8120 can be found here I2C Communication with the Honeywell HumidIcon Digital Humidity/Temperature Sensors (Version: 1.0)

The implementation works with interrupt, meaning that there are no busy-waiting involved.
